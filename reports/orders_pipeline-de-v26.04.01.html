CRITICAL   Actor: Transform Engine

Cleaned, validated Silver layer produced from Bronze with DQ checks enforced and PII masked.

AttributeDetail
PreconditionsBronze layer populated by UC-01.
PostconditionsSilver layer contains clean, typed, deduped, PII-masked records with DQ score ≥ threshold.

Flow Steps:

  1. Read from Bronze layer
  2. Apply column normalisation (rename, type cast, date format)
  3. Remove duplicates
  4. Apply null handling rules
  5. Mask PII columns (SHA256)
  6. Run DQ checks
  7. DQ FAIL → halt, alert, quarantine
  8. DQ PASS → write to Silver
